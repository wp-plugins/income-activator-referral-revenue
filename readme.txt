Add an Income Activator Referral Form to your WordPress website. An Income Activator account is required to use this plug-in.

== Description ==

This plug-in allows you to add an Income Activator Referral Form to your WordPress website.

It converts script code to short code that WordPress users can easily add to their page or theme.

Income Activator is a website builder that has software that allows users to send leads to companies that pay you.

You can create a custom Referral Form. You choose the companies you want to send referrals to.

You set the cost per lead. Leads are tracked and invoices are automatically created.

An Income Activator account is required to use this plug-in. 

== Installation ==

1. Upload the 'IncomeActivator' folder to the '/wp-content/plugins/' directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Create a Referral Form on your IA account.

4. Copy the Embed Form Code from your IA Referral Form.

5. Paste the code into the IA Referral Form Code box.

6. Click Save Changes.

7. You have a choice of adding your Referral Form to a website page, or to your Theme.

A. Add To A Page: Copy the Form Short Code into your website page, and the form will show up when you view your page.

B. Add To Theme: Copy the Theme Short Code into your website page, and the form will show up within your theme.

== Frequently Asked Questions ==

= What is Income Activator? =

Income Activator is an all inclusive website builder that includes 10 revenue streams such as referral programs, an online store,
and membership features. Income Activator is $30 per month, which includes access to the referral system, website hosting, and support.

= What do you make money from referrals? =

You can send visitor leads to companies that pay you for each lead. Leads can be sent through clicks, forms, or directory forms.
You find the company, form a relationship, and agree on a price per lead. Income Activator tracks the leads, notifies the company,
and creates an invoice that you can send to them.

= What is required to use this plug-in? =

You need an Income Activator account, an advertiser willing to pay you for leads, a referral program setup on Income Activator,
and a referral form connected to your program.

You also need to setup a landing page that your website visitor will be sent to after they complete the form.
